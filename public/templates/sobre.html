Sobre o Website

  • Este website foi desenvolvido no âmbito da disciplina de Desenvolvimento Web da Universidade Autónoma de Lisboa.
  • O Objetivo é usar uma API, criada também no contexto desta atividade, para colecionar dados e entrega-los a nós "clientes". Na teoria, o backend deste website recebe um pedido do cliente, um pedido de fetch por algum tipo de dados, é realizado no servidor a procura dos dados e entregue ao "cliente" no formato de JSON. Onde depois, é transferida a informação para o HTML com recurso a Javascript.
  • Nesta atividade para o backend foram utilizadas as bibliotecas Axios e Cheerio, foram instaladas com o comando npm install axios e npm install Cheerio. Além disso para ajudar na produtividade foi utilizado o nodemon.js que permite criar um servidor dinâmico que está sempre atento a mudanças nos ficheiros, de tal forma que torna a dinâmica de programação melhor.
  • O website consiste em explorar um pouco de um tema atual, o preço dos combustíveis. para visualizar melhor a proposta é um website que junte os dados sobre os preços das bombas de combustíveis da BP e Galp de Lisboa. Ao entrar na página inicial o utilizador pode escolher entre visualizar uma lista das bombas da BP ou da Galp, após selecionar uma bomba de combustível o utilizador é redirecionado para uma página que mostra o horário da Bomba selecionada, a morada e os preços atualizados mais recentes.
  • Na barra de navegação sempre presente no topo da página web, o utilizador pode carregar em "Combustíveis em Lisboa" e voltar à página principal. Ao clicar em comparador BPcomparador Galp, ou BP vs Galp o utilizador é redirecionado para uma página com gráficos que foram desenhados com recurso à biblioteca de front-end de javascript chart.js. Os gráficos apresentam os preços lado a lado de cada bomba de combustível da BP ou da Galp. Para melhor visualização ao clicar nas lables coloridas dos gráficos as linhas dos mesmos são anuladas temporariamente para melhor comparação.
  • A estética do site foi desenvolvida com recurso à framework de CSS Bootstrap, que ajuda imenso o desenvolvedor do front-end com classes de elementos HTML predefinidas. Para maior customização pequenas coisas foram alteradas relativamente ao Bootstrap e criadas no ficheiro my_styling.css.
